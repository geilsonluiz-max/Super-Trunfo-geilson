# Super Trunfo – Nível Aventureiro

Implementação do jogo Super Trunfo em linguagem C.

## Funcionalidades
- Menu interativo com switch/case
- Comparação de atributos entre duas cartas
- Regra especial: menor densidade demográfica vence
- Tratamento de opção inválida

## Como compilar
gcc super_trunfo.c -o super_trunfo

## Como executar
./super_trunfo
